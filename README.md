# LingRing

一款可在網路不穩定時自動切換到傳統撥號方式的網路電話應用程式，作者在開始動工此 App 時是使用預付卡，每個月除了網路費用 90 元並不希望額外付出通話費，打電話是很貴的服務，但生活中有時要聯絡廠商、訂便當等，因此若是對方有連上網的話，那我們就使用網路電話撥號，若否則立刻轉用傳統撥號方式，此 App 重點在於自動切換，節省時間。
